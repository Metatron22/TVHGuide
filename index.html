Tvhguide

An Android client for TVHeadend using the HTSP protocol

Download this project as a .zip file Download this project as a tar.gz file

TVHGuide is an Android client for TVHeadend, a TV streaming server. Some of its featrues are

  • List channels and their upcoming events
  • Search the EPG for future programmes
  • Schedual, cancel and remove recordings
  • Stream audio and video
